# Health Encryption System

This repository is a React frontend scaffold for the healthcare records platform powered by ABAC-based access control and encrypted cloud storage.

## What’s included

- React + Vite + TypeScript starter app
- ESLint configuration for TypeScript + React
- Base shell component with waiting state for the Figma design

## Next steps

1. Open the Figma design file.
2. Extract UI components and build the component library first.
3. Implement screens using the Figma frames as the single source of truth.

## Available scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run lint`
